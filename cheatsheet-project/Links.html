HTML Links Cheatsheet

Syntax

The HTML <a> tag defines a hyperlink, the syntax is:
<a href="url">text used for link"</a>

HTML Target Attributes

Attributes Value Description
target _self Default. Opens the document in the same window/tab as it was clicked
target _blank Opens the document in a new window or tab
target _parent Opens the document in the parent frame
target _top Opens the document in the full body of the window

URLs

There are two types of urls that can be used within links on HTML.
Absolute URLs: link to a Webpage:

<a href="https://www.codecademy.com">Codecademy</a>

Relative URLs: link to a local page

<a href="/resources/css/index.css">Index CSS file</a>
